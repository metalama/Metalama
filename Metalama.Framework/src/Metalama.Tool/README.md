## About

Metalama Command Line Tools is used for registering a Metalama license key, configuration, and troubleshooting.

## Commands

* `metalama license`: manage license registrations
* `metalama config`: edit or reset JSON configuration files
* `metalama cleanup`: delete temporary files
* `metalama kill`: shuts down or kills processes that may be locking Metalama packages
* `metalama telemetry`: manages the telemetry options and upload queue
* `metalama docs`: opens the Metalama documentation
* `metalama ui`: opens the configuration browser-based UI
* `metalama divorce`: copies code generated by Metalama back into your source code, in case you wanted to stop using Metalama
